# FQC Audit Tracker
**Alkhorayef Petroleum Company — GOS Operations**

A field quality audit tracking and KPI dashboard for the FQC (Field Quality Coach) program across all regions (Kuwait, Middle East, LATAM, Africa).

## Access
🔗 **Live dashboard:** https://engmmmk-oil.github.io/fqc-audit-tracker/

## Access levels
- 👁 **Viewer:** `TechHub26-view`
- ✎ **Editor:** `TechHub26`

## Features
- Audit entry and corrective action tracking
- KPI dashboard with traffic light indicators
- Advanced analytics by country, region, auditor
- Year vs Year comparison
- Value added tracking (Cost saved, NPT, Oil production, SWA)
- Export to CSV and Excel

## Regions covered
Kuwait · Middle East (SA, Oman, Iraq, UAE) · LATAM (Colombia, Mexico, Ecuador, Venezuela) · Africa (Egypt, Sudan, Libya, Congo)
